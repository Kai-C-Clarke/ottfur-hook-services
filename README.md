# Ottfur Hook Services

**Cair Aviation CW300 and CW400 Quick Release Hook — Calibration, Certification and Technical Documentation**

Operated by Jon Stiles, I/C1408, Little Millham, Robertsbridge, East Sussex.

---

## About Ottfur Hook Services

Ottfur Hook Services provides calibration, refurbishment and supply of Cair Aviation CW300 and CW400 glider tow release hooks. All work is carried out in accordance with Cair Aviation Pre-Delivery Test Procedures using the original Cair factory test rig. Certified units are issued with a completed First Refurb Test Certificate and updated Log Book.

## Test Standards

All calibration work is performed against the following parameters:

| Parameter | Specification |
|-----------|--------------|
| No-load lever force | 2.5 daN ±1.0 daN (5.5 lbs ±2.2 lbs) |
| ISU Datum (2kN, 0°) | 6.9 daN ±0.2 daN (15.5 lbs ±0.5 lbs) |
| Load test band (all angles) | 2.5 daN min — 20 daN max (5.5–44 lbs) |
| Back release breakout | 3.0 daN ±1.0 daN (6.6 lbs ±2.2 lbs) |
| Back release full travel | 6.0 daN ±2.0 daN (13.25 lbs ±4.4 lbs) |
| Back release angle | 85° ±5° |
| Cable tension tolerance | ±5% |
| Cable angle tolerance | ±3° |

Test rig: Cair Aviation factory original. Long lever CW302 (68.5mm outer hole) used for all ISU and load tests. Digital peak-hold gauge (500N) used for all force measurements.
